Button Methods

The toggle method makes the button look pressed. Click on both buttons, but also click outside the button afterwards to see the difference.

The loading method disables the button and changes the button text to "loading...".

The reset method changes the button text to its original text (if changed).

The string method changes the button text.

Note: You must add the data-STRING-text attribute with a value of the specified button text, for this example to work.
