Navbar Link

The .navbar-link class styles an element to look like a link inside the navbar (anchors get proper padding and an underline on hover, while other elements like p or span gets a default hover effect - white color in an inversed navbar and a black color in a default navbar).

Hover over the a and the span element to see the effect.
