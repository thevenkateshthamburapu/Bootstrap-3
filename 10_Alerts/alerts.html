Alerts

Alerts are created with the .alert class, followed by one of the four contextual classes .alert-success, .alert-info, .alert-warning and .alert-danger

Info! This alert box could indicate a neutral informative change or action.
Warning! This alert box could indicate a warning that might need attention.
Danger! This alert box could indicate a dangerous or potentially negative action.

Alert Links

Add the alert-link class to any links inside the alert box to create "matching colored links".

Closing Alerts

The a element with class="close" and data-dismiss="alert" is used to close the alert box.

The alert-dismissible class adds some extra padding to the close button.
