Tooltip Options

animation

The animation option specifies whether to add a CSS fade transition effect to the tooltip.

container

The container option appends the tooltip to a specific element.

delay

The delay option specifies the number of milliseconds it will take to show and hide the popover.

This tooltip will take 1000 milliseconds to show when you hover over it, and 1000 milliseconds to hide on mouse leave.



This tooltip will take 500 milliseconds to show when you hover over it, and 1000 ms to hide on mouse leave.

html

The html option specifies whether to accept HTML tags in the tooltip.

placement

The placement option specifies the tooltip position.

selector

The selector option adds a tooltip to a specified selector.

title

The title option specifies the text that should be displayed inside the tooltip.

trigger

The trigger option specifies how the tooltip is triggered.

Tooltip Methods

.tooltip(options)

The placement option specifies the tooltip position.

Tooltip Example

The show.bs.tooltip event occurs when the tooltip is about to be shown.

The shown.bs.tooltip event occurs when the tooltip is fully shown.

The hide.bs.tooltip event occurs when the tooltip is about to be hidden.

The hidden.bs.tooltip event occurs when the tooltip is fully hidden.
