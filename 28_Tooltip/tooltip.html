Tooltip Options

data-animation

The data-animation option specifies whether to add a CSS fade transition effect to the tooltip.

data-container

The data-container option appends the tooltip to a specific element.

data-delay

This tooltip will take 1000 milliseconds to show when you hover over it, and 1000 milliseconds to hide on mouse leave.

data-html

The data-html option specifies whether to accept HTML tags in the tooltip.

data-placement

The data-placement option specifies the tooltip position.

data-selectors

The data-selectors option adds a tooltip to a specified selector.

data-title

The data-title option specifies the text that should be displayed inside the tooltip.

data-trigger

The data-trigger option specifies how the tooltip is triggered.
