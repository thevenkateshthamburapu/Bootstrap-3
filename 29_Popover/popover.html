Popover Options

data-animation

The data-animation< option specifies whether to add a CSS fade transition effect when opening and closing the popover.

data-container

The container option appends the popover to a specific element.

data-delay

The delay option specifies the number of milliseconds it will take to open and close the popover.

This popover will take 1000 milliseconds to open and close.

data-content and data-title

The data-title option specifies the header text of the popover.

The data-content option specifies the text that should be displayed inside the popover's body.

data-html

The data-html option specifies whether to accept HTML tags in the popover.

data-placement

The placement option specifies the popover position.

data-selectors

The selector option adds a popover to a specified selector.

data-trigger

The data-trigger option specifies how the popover is triggered.
