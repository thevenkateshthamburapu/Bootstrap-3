Popover Options

animation

The animation option specifies whether to add a CSS fade transition effect when opening and closing the popover.

container

The container option appends the popover to a specific element.

content and title

The title option specifies the header text of the popover.

The content option specifies the text that should be displayed inside the popover's body.

delay

The delay option specifies the number of milliseconds it will take to open and close the popover.

This popover will take 1000 milliseconds to open and close.



This popover will take 500 milliseconds to open, and 1000 ms to close.

html

The html option specifies whether to accept HTML tags in the popover.

placement

The placement option specifies the popover position.

selector

The selector option adds a popover to a specified selector.

trigger

The trigger option specifies how the popover is triggered.

Popover Methods

.popover(options)

The placement option specifies the popover position.

Popover Events

Popover Example

The show.bs.popover event occurs when the popover is about to be shown.

The shown.bs.popover event occurs when the popover is fully shown.

The hide.bs.popover event occurs when the popover is about to be hidden.

The hidden.bs.popover event occurs when the popover is fully hidden.
